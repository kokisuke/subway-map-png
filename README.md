# インタラクティブ名古屋地下鉄路線図

このプロジェクトは、名古屋の地下鉄路線図をインタラクティブに表示するウェブアプリケーションです。

## 機能

- **路線図の表示**: 名古屋の地下鉄路線図（`map.png`）を表示します。
- **駅情報の表示**: 路線図上の駅をクリックすると、画面右側にその駅の詳細（路線、乗り換え情報など）が表示されます。
- **スクロール可能な地図**: 地図が大きい場合でも、コンテナ内で地図をスクロールして全体を見ることができます。

## 使い方

1.  リポジトリにある `index.html` ファイルをウェブブラウザで開きます。
2.  左側に表示された地図上の駅をクリックします。
3.  右側のパネルに駅の詳細情報が表示されます。

## 駅の追加・更新方法

駅の情報を追加または更新するには、`script.js` ファイル内の `stations` 配列を編集します。

1.  画像編集ソフトなどを使って `map.png` を開き、追加したい駅のピクセル座標（X, Y）を特定します。
2.  `script.js` を開き、`stations` 配列に新しい駅オブジェクトを追加します。

```javascript
const stations = [
    // ... 既存の駅データ
    {
        name: '新しい駅名',
        x: 123, // 特定したX座標
        y: 456, // 特定したY座標
        lines: ['路線名1', '路線名2'],
        details: '駅に関する詳細情報'
    }
];
```
